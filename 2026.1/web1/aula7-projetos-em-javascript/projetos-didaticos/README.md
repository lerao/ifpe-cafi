# 10 Mini-Projetos JavaScript para Iniciantes

Este repositório contém uma coleção de 10 projetos práticos desenvolvidos para facilitar o ensino de JavaScript puro (Vanilla JS) para turmas de iniciantes.

## 🚀 Como usar este material
1.  Abra a pasta `projetos-didaticos`.
2.  Abra o arquivo `index.html` principal em seu navegador para ver o dashboard de todos os projetos.
3.  Cada pasta de projeto contém um arquivo `explicacao.md` detalhando os conceitos de programação utilizados naquele código específico.

## 🛠️ Tecnologias Utilizadas
-   **JavaScript (ES6+)**: Foco total em clareza, nomes de funções intuitivos e comentários didáticos.
-   **Bootstrap 5**: Para garantir que os projetos tenham um visual moderno, profissional e responsivo sem a necessidade de escrever CSS complexo.
-   **Bootstrap Icons**: Para ícones intuitivos nas interfaces.

## 📚 Lista de Projetos e Níveis
1.  **Calculadora de Média**: Iniciante (Variáveis e Condicionais simples)
2.  **Calculadora de IMC**: Iniciante (Else if e Validações)
3.  **Calculadora de Desconto**: Iniciante (Matemática e Formatação)
4.  **Teste de Aptidão**: Iniciante (Operadores Relacionais)
5.  **Contador**: Intermediário-Iniciante (Estado e Eventos múltiplos)
6.  **Tabuada**: Intermediário-Iniciante (Laços de Repetição)
7.  **Financiamento**: Intermediário-Iniciante (Lógica de Negócio)
8.  **Lâmpada**: Intermediário-Iniciante (Manipulação de DOM/Atributos)
9.  **Semáforo**: Intermediário (Temporizadores e Arrays)
10. **Todo-List**: Avançado-Iniciante (Criação Dinâmica de Elementos)
11. **Relógio com Alarme**: DESAFIO (Objeto Date, Timers e Áudio)
