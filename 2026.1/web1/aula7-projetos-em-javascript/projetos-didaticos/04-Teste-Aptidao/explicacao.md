# Explicação: Teste de Aptidão Física

Este projeto ensina como comparar valores numéricos usando operadores lógicos e relacionais.

## Conceitos de JavaScript utilizados:

1.  **Operador Relacional `<= ` (Menor ou Igual)**: Diferente de outros projetos onde buscamos valores maiores (como notas), aqui a regra de negócio exige que o valor seja menor que o limite.
2.  **Constantes (`const`)**: Definimos o `TEMPO_LIMITE` como uma constante em letras maiúsculas (convenção para valores que nunca mudam no código).
3.  **Feedback Visual Imediato**: Usamos cores contrastantes (verde e vermelho) e emojis para que o usuário identifique o resultado instantaneamente.
4.  **Estilo Escuro com Bootstrap**: Aplicamos as classes `bg-dark` no body e `bg-secondary` no container para criar uma interface moderna e esportiva.
