# Validações Exemplificadas 

| Campo              | Validação                     |
| ------------------ | ----------------------------- |
| Input text         | tamanho mínimo                |
| Input email        | regex                         |
| Input CPF          | quantidade de dígitos         |
| Input number       | intervalo (18 a 120)          |
| Input date         | obrigatório                   |
| Select             | opção obrigatória             |
| Radio              | seleção obrigatória           |
| Checkbox múltiplo  | pelo menos uma opção          |
| Password           | tamanho mínimo                |
| Confirmar senha    | igualdade                     |
| Textarea           | mínimo e máximo de caracteres |
| Checkbox de aceite | obrigatório                   |
